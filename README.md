# Desafio #2 - Middlewares

Desafio #2 do curso Rocketseat Ignite Node.js.

## Pré-requisitos

* Node
* Yarn

## Comandos

Todos os comandos a partir do diretório raíz do projeto.

### Instalar Dependências

```bash
yarn install
```

### Rodar Localmente

```bash
yarn dev
```

### Testes

```bash
yarn test
```
